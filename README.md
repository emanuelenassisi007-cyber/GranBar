# GranBar Website

A modern and responsive website developed for **GranBar**, designed to provide customers with an elegant online presence and easy access to the bar's main information.

## Features

- Responsive design for desktop, tablet and mobile
- Home page with modern layout
- Gallery section
- Contact information
- Interactive map
- Optimized navigation
- Clean and minimal UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- GitHub Pages
